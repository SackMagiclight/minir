MinIR is...

It is IR designed for beatoraja.

How to Use

  1. Sign up
    https://www.gaftalk.com/minir/signup/ Create a user and click the link in the confirmation E-mail.

  2. Download the MinIR API
    https://drive.google.com/open?id=1wYVNufrfiVAWUkYTVdGl0Q0PqrhkHZBo Download this zip file.
    Its contents are as follows:

    • ir
      • minir_alpha.jar
    • beatoraja_minir.bat

    Extract the files to the root directory of beatoraja.
    It would look like this:

    • skin
    • player
    • ir
      • minir_alpha.jar
    • beatoraja_minir.bat
    • beatoraja.jar
    • beatoraja-config.bat
    • config.json
    • ...

    Please notice that although the IR function itself is independent and should not affect local data, it is strongly recommended to keep a backup.

  3. Run beatoraja_minir.bat.
    Select MinIR under IR tab in the configuration window. Fill the Username and Password used when signing up. Click Start.

Current Functions

  • Sending scores for BMS files
    • Scores will be sent automatically at the result screen.
    • Old scores will be overriden. Better clear lamps will be kept.
    • You must log in from beatoraja to send scores.
  • Sending scores for courses (for beatoraja 0.5.6 or later)
    • Better scores and clear lamps will be kept.
    • You must log in from beatoraja to send scores.
  • Checking Internet Ranking for BMS files
    • https://www.gaftalk.com/minir/viewer/?songhash=[sha256]
      Please replace [sha256] with sha256 of the BMS file.
      • Add parameter lnmode=0,1,2 to check scores in different LN mode. (0:LN, 1:CN, 2:HCN)
  • Checking Internet Ranking for courses
    • https://www.gaftalk.com/minir/viewer/cource?courcehash=[sha256]
      Please replace [sha256] with sha256 of the course file.
  • Checking the most recent played 100 charts
    • https://www.gaftalk.com/minir/viewer/latest?mode=[mode]
    • Replacing parameter mode=[mode] with indicators to check different modes. (Default is 7KEYS(beat-7k))
      • [mode] indicators
        • beat-5k : 5KEYS
        • beat-7k : 7KEYS
        • beat-10k : 10KEYS
        • beat-14k : 14KEYS
        • popn-5k : PMS 5KEYS
        • popn-9k : PMS 9KEYS
        • keyboard-24k : KM 24KEYS
        • keyboard-24k-double : KM 48KEYS
  • Checking the most recent played 100 courses
    • https://www.gaftalk.com/minir/viewer/cource_latest
