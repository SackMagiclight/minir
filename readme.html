MinIR とは

beatoraja用に設計されたIRです。 下記コンセプトに基づいています。

  • できるだけ安価に済ませたい
    • なので、AWSの無料枠で収まるように設計
    • 保存するデータを最小限にすることで、課金を抑える
      • 保存データが確定するまでもうちょっとかかるかな？
  • 最低限の機能
    • スコアが譜面ごとに記録できることのみを要件に開発
    • そのため下記内容は実装予定がありません（今後変わるかもだけど）
      • コースIR 実装予定です
      • 段位認定（難易度表で対応してください。スコアは今後送信できるようにします。）
      • ライバル機能（無課金で対応するのが難しい）
      • マイページ（同上）
      • 楽曲検索(RDBじゃないので非現実的な負荷)
  • 万が一に備えた再利用性
    • AWSを利用することで、サーバー管理の引き継ぎが容易
  • ぶっちゃけAWSの勉強で作った
    • すいません、ガッツリ運用するかは未知数です
    • 人が足りない
    • 本当は実装予定なし機能はやりたい…
    • 規模的にDiscode IRに改名しても良いかも…

参加方法

  1. IRユーザーを作成します。
    https://www.gaftalk.com/minir/signup/ にアクセスして、ユーザーを作成し、メール認証を行います。

  2. MinIR API をダウンロードし配置します
    https://drive.google.com/open?id=1wYVNufrfiVAWUkYTVdGl0Q0PqrhkHZBo からzipをダウンロードします。
    フォルダ内容は以下の通りです。

    • ir
      • minir_alpha.jar
    • beatoraja_minir.bat

    これらのファイルを本体であるbeatorajaフォルダに配置します。
    以下のようになるはずです。

    • skin
    • player
    • ir
      • minir_alpha.jar
    • beatoraja_minir.bat
    • beatoraja.jar
    • beatoraja-config.bat
    • config.json
    • ...

    注意：IR機構は独立しているため、ローカルのプレイデータに影響は与えないはずですが、万が一のためバックアップをとっておくことを強く推奨します。

  3. beatoraja_minir.bat から起動します。
    IRタブで「MinIR」が選択できるようになっていれば成功です。 登録したメールアドレス/パスワードを入力し、起動してください。

現状できること

  • 譜面のスコア送信
    • IR設定で「常時」「完走時」「更新時」がありますが、現状IR送信されたら全部上書きしているので「更新時」を強く推奨します。
      • スコアが上回っていれば上書き、クリアが上回っていれば上書きされます。
    • beatoraja上でのログイン必須です。
  • 譜面のスコア閲覧
    • https://www.gaftalk.com/minir/viewer/?songhash=[sha256] で閲覧できます。[sha256]には譜面のsha256ハッシュが必要です。
      • 現状、譜面のsha256ハッシュの取得がハードル高めです。
      • なんとかしたいのはしたいのですが…　良い意見募集中。
    • ログインは不要です。
