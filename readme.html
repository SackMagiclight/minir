MinIR とは

beatoraja用に設計されたIRです。 下記コンセプトに基づいています。

  • できるだけ安価に済ませたい
    • なので、AWSの無料枠で収まるように設計( 収まってほしい)
    • 保存するデータを最小限にすることで、課金を抑える
      • 保存データが確定するまでもうちょっとかかるかな？
  • 最低限の機能
    • スコアが譜面ごとに記録できることのみを要件に開発
    • そのため下記内容は実装予定がありません（今後変わるかもだけど）
      • 段位認定（難易度表で対応してください。スコア送信できます。(0.5.6以降の本体必須)）
      • ライバル機能（無課金で対応するのが難しい）
      • 楽曲検索(RDBじゃないので非現実的な負荷)
  • 万が一に備えた再利用性
    • AWSを利用することで、サーバー管理の引き継ぎが容易
  • ぶっちゃけAWSの勉強で作った
    • すいません、ガッツリ運用するかは未知数です
    • 人が足りない
    • 本当は実装予定なし機能はやりたい…

参加方法

  1. IRユーザーを作成します。
    https://www.gaftalk.com/minir/signup/ にアクセスして、ユーザーを作成し、メール認証を行います。

  2. MinIR API をダウンロードし配置します
    https://github.com/SackMagiclight/minir/releases/download/20180714/minir_beta1.zip からzipをダウンロードします。
    フォルダ内容は以下の通りです。

    • ir
      • minir_alpha.jar
    • beatoraja_minir.bat

    これらのファイルを本体であるbeatorajaフォルダに配置します。
    以下のようになるはずです。

    • skin
    • player
    • ir
      • minir_alpha.jar
    • beatoraja_minir.bat
    • beatoraja.jar
    • beatoraja-config.bat
    • config.json
    • ...

    注意：IR機構は独立しているため、ローカルのプレイデータに影響は与えないはずですが、万が一のため バックアップをとっておくことを強く推奨します。

  3. beatoraja_minir.bat から起動します。
    IRタブで「MinIR」が選択できるようになっていれば成功です。 登録したメールアドレス/パスワードを入力し、起動してください。

現状できること

  • 譜面のスコア送信
    • スコアが上回っていればスコアが上書き、クリアが上回っていればクリア状況が上書きされます。
    • beatoraja上でのログイン必須です。
  • コースのスコア送信(beatoraja 0.5.6以上必須)
    • スコアが上回っていればスコアが上書き、クリアが上回っていればクリア状況が上書きされます。
    • beatoraja上でのログイン必須です。
  • 譜面のスコア閲覧
    • https://www.gaftalk.com/minir/viewer/?songhash=[sha256]&lnmode=[0,1,2] で閲覧できます。[sha256]には譜面のsha256ハッシュが必要です。
      • クエリパラメータ lnmode=0,1,2 でLNモード別のスコアが閲覧できます。(0:LN, 1:CN, 2:HCN)
        • 指定がない場合、LNとなります
      • BMSに #LNMODEが定義されている場合は常に lnmode=0となるので、注意してください。（LNモードにかかわらずユニークのためです）
  • コースのスコア閲覧
    • https://www.gaftalk.com/minir/viewer/cource?courcehash=[sha256] で閲覧できます。[sha256]にはコースのsha256ハッシュが必要です。
  • 直近100件のIR送信された譜面リスト
    • https://www.gaftalk.com/minir/viewer/latest?mode=[mode] で閲覧できます。
    • クエリパラメータ mode=[mode] でモード別の直近が取得できます（指定なしで7KEYS (beat-7k))
      • [mode]一覧
        • beat-5k : 5KEYS
        • beat-7k : 7KEYS
        • beat-10k : 10KEYS
        • beat-14k : 14KEYS
        • popn-5k : PMS 5KEYS
        • popn-9k : PMS 9KEYS
        • keyboard-24k : KM 24KEYS
        • keyboard-24k-double : KM 48KEYS
  • 直近100件のIR送信されたコースリスト
    • https://www.gaftalk.com/minir/viewer/cource_latest で閲覧できます。
  • ユーザーマイページ
    • https://www.gaftalk.com/minir/viewer/user?userid=[userid] で閲覧できます。
      • クエリパラメータ userid=[userid] が必要です。
      • IRページのユーザー名クリックから遷移するのが確実です。
  • ログイン
    • https://www.gaftalk.com/minir/user/login で開きます。
    • 登録時のメールアドレス、パスワードを利用してログインしてください。
    • ログイン後はユーザーマイページ（編集可能版）に遷移します。（後述）
  • ユーザーマイページ（編集）
    • ログインが正しく行われていれば、 https://www.gaftalk.com/minir/user/ で開きます。（ログイン後は自動遷移します。一定時間はログイン情報を保持します。）
    • bioの編集が可能です。
      • URL（http:// | https:// 等）はユーザーマイページ内でリンクに置き換えられるため、外部リンクにご活用ください。

禁止事項

  • 違法な楽曲の登録・プレイ
    • 発見次第、プレイアカウントは永久凍結処分となります。
  • ユーザコメント欄における誹謗中傷
    • 管理者裁定の無い限り削除しません。また管理者には削除の義務や責任はありません。
    • 個人を完全に特定する情報を伴っているものは削除対象です。
  • ユーザコメント欄における私生活情報の記述
    • 情報価値が無く、私事のみの情報・第三者の確認できないプライベート情報は、個人が完全に特定されなくても、対象者に不利益が発生する可能性があれば、一律削除対象とします。
  • ユーザコメント欄における外部リンク
    • 営利目的での外部リンク（アフィリエイトサイトなど）、悪意あるサイトへのリンク（フィッシングサイトなど）は削除対象です。
    • 非営利目的での外部リンク（外部SNS、他BMSリポジトリへのリンク等）は削除対象外です。
